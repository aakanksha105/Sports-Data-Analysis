# Data Analysis – Indian Premier League (IPL)

An exploratory data analysis (EDA) project focused on analyzing Indian Premier League (IPL) match data to identify successful teams, key players, and factors contributing to match outcomes. The project also derives insights to support sports marketing and endorsement decisions.

---

## Problem Overview

The objective of this project is to perform exploratory data analysis on Indian Premier League (IPL) data to answer key analytical questions, including:

- Identifying the most successful teams and players in the league
- Understanding factors that contribute to a team’s win or loss
- Analyzing performance trends across seasons
- Suggesting teams or players suitable for brand endorsements based on performance and consistency

The analysis is data-driven and focuses on uncovering meaningful patterns from historical IPL match data.

---

## Dataset

The dataset contains detailed IPL match and ball-by-ball information, including:
- Team and player statistics
- Match results and outcomes
- Performance metrics across seasons

Data source:
https://bit.ly/34SRn3b

---

## Analysis Workflow

- Loaded and validated IPL match and delivery datasets
- Cleaned and preprocessed missing and inconsistent values
- Performed exploratory data analysis to study team and player performance
- Analyzed factors influencing match outcomes such as toss decisions, venue, and scoring patterns
- Visualized trends using statistical plots and comparative charts

---

## Key Insights

The analysis highlights:
- Teams with consistently strong performance across seasons
- Players with high impact on match outcomes
- Factors such as venue, toss decisions, and batting order influencing wins and losses
- High-visibility teams and players suitable for brand endorsements

These insights demonstrate how sports data can be leveraged for performance analysis and marketing strategy.
